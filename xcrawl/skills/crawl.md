# XCrawl Crawl — 批量爬取

## 描述

从URL列表批量抓取多个网页。

## 输入

用户指定URL列表文件路径（每行一个URL），或直接传入URL数组。

## 执行

1. 读取URL列表
2. 逐个调用XCrawl抓取
3. 保存到 `raw/articles/`
4. 记录抓取结果（成功/失败）

## 输出




raw/articles/
├── {日期}-{标题1}.md # 成功
├── {日期}-{标题2}.md # 成功
└── crawl-log-{日期}.md # 抓取日志（失败记录）

## 错误处理
- 单个URL失败不影响其他
- 失败URL记录到 `crawl-log-{日期}.md`
- 最终报告：成功数/总数
