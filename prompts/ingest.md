# 摄入流程提示词

原始资料进入 raw/ 后的处理流程。

## 支持的输入类型

| 类型 | 存放位置 |
|------|---------|
| XCrawl 抓取的网页 | raw/articles/ |
| URL 列表 | raw/urls/ |
| 本地 .md/.txt 文件 | raw/notes/ |
| 飞书文档 | raw/feishu_docs/ |

## 执行流程

1. 读取文件，理解内容类型
2. 自动分类到对应子目录
3. 生成简要入库记录（文件名、来源、日期）
4. 追加到 `wiki/indexes/log.md`

## 隐私过滤

入库前检查以下内容，标记为"内部使用"不同步飞书：
- 涉及个人隐私的讨论
- 团队内部未公开的计划
- 财务或人事敏感信息

## 格式标准化

非 .md 文件统一转为 UTF-8 Markdown，清理乱码和格式噪音。
